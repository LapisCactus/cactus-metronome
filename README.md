# cactus-metronome
This is a repository of CactusMetronome, a metronome library for android.

CactusMetronome

## 概要 ##

Android向けのメトロノームライブラリです。

## 使用方法 ##

cactusmetronomelib.jarをプロジェクトに取り込みます。
詳細は準備中ですが、サンプルアプリをご参照ください。

## ビルド方法 ##

1. コマンドプロンプトで"gradlew clean build makeJar"を実行します。
2. ライブラリが、cactusmetronomelib/release/cactusmetronomelib.jarとして生成されます。
3. サンプルアプリが、app/build/outputs/apk/app-debug.apkとして生成されます。
